Coloca aquí los GIFs adjuntos para que aparezcan como fondos animados permanentes.

Nombres de archivo esperados:
- `florecita.gif`  -> imagen de la flor (primera imagen adjunta)
- `chillmode.gif`  -> imagen de la habitación/cama (segunda imagen adjunta)

Instrucciones:
1. Descarga las imágenes adjuntas desde la conversación.
2. Guarda los GIFs en esta carpeta `assets/` con los nombres indicados arriba (`florecita.gif` y `chillmode.gif`).
3. Recarga `paginaprincipal.html` en el navegador.

Notas:
- Estos fondos se agregaron como entradas por defecto en `app.js` con `isDefault: true`, por lo que no podrán eliminarse desde la UI (solo los fondos personalizados pueden borrarse).
- Si prefieres otros nombres o ubicaciones, actualiza las rutas en `app.js` dentro de `defaultBackgrounds`.
